# Mini Store App

The "Mini Store" app is a clean, modern, and responsive e-commerce UI prototype developed with **Flutter**. This project is designed to provide a seamless user experience for browsing a curated list of products with a minimalist aesthetic.

## 🚀 Overview
This application is built with a strong emphasis on **Clean Code Architecture**. It focuses on maintainability, readability, and scalability, ensuring that the codebase is well-structured for future enhancements or API integrations.

### Key Features:
- **Modern UI/UX:** A sleek design featuring a professional color palette (Blue, White, and Grey).
- **Modular Architecture:** Logic and UI are separated using **Reusable Widgets** for efficient code management.
- **Cross-Platform Compatibility:** Optimized for both Android and iOS devices.
- **Responsive Layout:** Utilizes `GridView.builder` to ensure a consistent visual experience across various screen sizes.

## 🛠 Tech Stack
- **Framework:** Flutter SDK
- **Language:** Dart
- **Development Environment:** VS Code
